These scripts can be used to interface with the HUE API to controll your hue bulbs.  It requires you to generate a User ID for the hue API as well as jq 1.5.
jq: https://stedolan.github.io/jq/download/
hue api user: https://www.developers.meethue.com/documentation/getting-started
This also Requires HAMBridge: http://solutionsetcetera.com/about-ham-bridge.html
