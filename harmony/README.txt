Hubitat Harmony is a SmartApp give users the ability to control each device by Logitech Harmony. It works based on 'Harmony API'
https://github.com/maddox/harmony-api

This is a port from KuKu Harmony to work on the Hubitat home automation platform.

With Hubitat Harmony based on 'Harmony API', you can turn on and turn off devices individually. The device's other commands are also support.

ex) volume up/down, channel up/down, FanSpeed, Menu and so on

You will need to install both smart apps and all of the device drivers for this integration to work.  You also need to have Harmony API running somewhere.
